Bemvindo ao Slide!

Um modelo de apresentação de slides usando apenas HTML5 e CSS3, formatado utilizando Markdown.

  • apresenta.html: arquivo onde ficam os tags markdown
  • include/icon.png: logo que vai aparecer em todos os slides (256x256)
  • include/splash.png: imagem que será apresentada como tela de abertura (1024 x 768)
