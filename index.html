Bemvindo ao Apresenta!

O projeto é um simples modelo usando HTML5 e CSS3 que é apresentado usando uma biblioteca Javascript chamada Remark. Esta biblioteca gera conteúdo usando Markdown como lightweight markup language.

  • modelol.html: arquivo onde está os tags markdown
  • include/icon.png: logo que vai aparecer em todos os slides (256x256)
  • include/splash.png: imagem que será apresentada como tela de abertura (1024 x 768)
